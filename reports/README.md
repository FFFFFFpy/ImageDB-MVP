# Reports output directory

This root `reports/` directory is reserved for generated local reports from scripts such as:

- `scripts/check-env.ps1`
- `scripts/check-env.sh`
- `pnpm release:performance`

Historical MVP1 reports have been hard-archived under:

```text
docs/MVP1/archive/reports/
```

Current MVP1 status is documented under:

```text
docs/MVP1/
```

Do not use this root directory as the current project-status source. It is an output location for newly generated diagnostic or gate reports.
