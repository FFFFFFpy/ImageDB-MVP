# M6.5 Definition of Done

状态说明：本清单用于发布复验。仓库已记录本地 PostgreSQL 18.4 + pgvector 0.8.3 真实验证通过，但发布级 Windows 安装包验收仍需在 M9 重新执行。

- [ ] 安装包包含固定版本 PostgreSQL 与 pgvector。
- [ ] 运行时 manifest 和文件 Hash 校验生效。
- [ ] 正式模式只使用 resource_dir 运行时。
- [x] 用户数据不写入安装目录。（已由托管数据库应用数据布局与真实运行验证覆盖，发布包仍需复验。）
- [ ] 干净 Windows 无需安装 PostgreSQL。
- [x] PostgreSQL 仅监听 loopback。（本地托管运行时验证已覆盖，发布包仍需复验。）
- [x] 首次初始化、重试和崩溃恢复通过。（真实 PostgreSQL 与恢复测试已覆盖，发布包仍需复验。）
- [x] stale PID、端口冲突和多实例保护通过。（真实/故障测试已覆盖相关恢复路径，发布包仍需复验。）
- [x] vector extension 与 migration 健康检查通过。（PostgreSQL 18.4 + pgvector 0.8.3 已验证。）
- [ ] 备份和恢复通过。
- [ ] 非 ASCII 路径通过。
- [ ] Windows 安装、升级、卸载行为通过。
