# Milestone 1：应用骨架与数据库

## 实现

1. 完成前端路由、布局、错误边界和 Query Client。
2. 完成 Commands、Services、Domain、Repositories、Infrastructure 分层。
3. 完成托管 PostgreSQL 生命周期。
4. 完成外部 PostgreSQL 连接测试。
5. 执行初始迁移。
6. 完成设置存储、凭据存储、日志和单实例控制。
7. 完成首次启动页和数据库设置页。

## 验收

- 首次启动完整完成数据库初始化。
- 重启后直接连接已有数据库。
- 数据库状态可在 GUI 查看。
- 外部连接可以执行版本、扩展和权限检查。
- 初始 Schema 与迁移测试通过。
